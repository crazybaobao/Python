运维

【shell】

  • 强制查杀应用进程：
    • sc queryex 应用名
    taskkill /pid xxx /f /t

【supervisor】

    supervisor启停：
  1. supervisorctl -u 用户名 -p 密码 start 应用名
  2. supervisorctl -u 用户名 -p 密码 stop 应用名

【nginx】
查看nginx进程是否存在：ps -ef | grep nginx
nginx配置语法检查：/usr/local/nginx/sbin/nginx -t
nginx重载：nginx -s reload

【docker】
查看镜像：docker images
删除镜像：docker rmi 【image id】
删除容器：docker rm 【container id】
查看所有容器：docker ps -a
docker启动tomcat：docker run --name tomcat -p 8888:8080 tomcat:8.0

【git】
查看当前git版本：git --version
强制拉取远程仓库覆盖本地仓库：
git fetch --all
git reset --hard origin/master
git pull
git fetch --all && git reset --hard origin/master && git pull（用一条命令）
